How the Calculation Works

The tool compares two future net-worth paths over the selected timeline. It does not try to predict the market perfectly; it makes the assumptions visible so you can stress-test the decision.

Sell now path

The sale path estimates the cash available after selling, then compounds that cash as if it were invested elsewhere.

  • Net sale proceeds = current value minus selling costs minus mortgage payoff.
  • Taxable gain = current value minus cost basis minus selling costs, never below zero.
  • After-tax proceeds are compounded annually using the investment return input.

Rent it out path

The rental path estimates annual cash flow, future equity, and what you could net if you sold at the end of the projection.

  • Annual cash flow = rent minus vacancy, percentage and fixed management fees, maintenance, recurring costs, mortgage payments, and rental tax.
  • Future equity uses projected home value minus the estimated remaining mortgage balance.
  • Final rental value = projected after-tax sale proceeds at the end plus cumulative rental cash flow.
